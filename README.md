static-files-manager
====================

## Descprition
Node.js application for managing static files

## Prerequisites
* `Node.js >= 7.4.0`
* `npm >= 3.10.5`
* `SQLite >= 3.14.1`

## Usage
* `gulp` - build and run the application
* `gulp test` - build and run all tests
* `gulp build` - create an application build
